## 🚀 技术实现亮点

### 1. 后端架构
- **框架**: FastAPI (Python)
- **数据库**: PostgreSQL + SQLAlchemy ORM
- **认证**: JWT + OAuth2 + RBAC
- **实时通信**: WebSocket
- **数据处理**: pandas + openpyxl
- **API文档**: Swagger UI / ReDoc

### 2. 前端架构
- **框架**: Flutter (跨平台 iOS/Android)
- **状态管理**: Provider
- **网络请求**: HTTP + WebSocket
- **本地存储**: SharedPreferences
- **图表**: FL Chart
- **动画**: 自定义弹幕效果

### 3. 数据可视化
- **折线图**: 个人成长曲线
- **柱状图**: 小组对比
- **环形图**: 积分使用情况
- **统计卡片**: 关键指标展示
- **实时动态**: 弹幕滚动效果

### 4. 用户体验
- **深色模式**: 赛博朋克风格
- **响应式设计**: 适配不同屏幕
- **交互反馈**: 动画和提示

## 🚀 部署和运行

### 环境要求
- Python 3.8+
- Flutter 3.0+
- PostgreSQL 12+
- Redis 6+ (可选)

### 快速启动
```bash
# 后端
cd backend
pip install -r requirements.txt
python init_db.py
uvicorn main:app --reload

# 前端
cd frontend
flutter pub get
flutter run
```

### 访问地址
- API服务: http://localhost:8000
- API文档: http://localhost:8000/docs
- WebSocket: ws://localhost:8000/api/v1/ws
