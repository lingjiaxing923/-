## 🔧 快速启动

### 后端
```bash
cd backend
pip install -r requirements.txt
python init_db.py
uvicorn main:app --reload
```

### 前端
```bash
cd frontend
flutter pub get
flutter run
```

### 访问
- API: http://localhost:8000
- 文档: http://localhost:8000/docs
