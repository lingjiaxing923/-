## 🚀 快速开始

### 前置条件
1. ✅ 安装 Flutter SDK (3.0.0+)
2. ✅ 安装 Android SDK (API 21+)
3. ✅ 配置 Java JDK 11+
4. ✅ 准备 Android 设备或模拟器

### 构建步骤
```bash
# 1. 进入项目目录
cd /workspaces/-/classquest

# 2. 运行构建脚本
./build_android.sh

# 3. 选择构建类型
# 选项1: Release APK (推荐用于测试)
# 选项2: App Bundle (用于Google Play发布)
# 选项3: Debug APK (用于开发调试)
# 选项4: 全部构建
```

### 手动构建 (可选)
```bash
# 1. 进入前端目录
cd frontend

# 2. 获取依赖
flutter pub get

# 3. 构建 Release APK
flutter build apk --release

# 4. 查找构建产物
ls -lh build/app/outputs/flutter-apk/app-release.apk
```
